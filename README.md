# Premium Digital Clock

## 📌 Description
A modern digital clock with animated background, glassmorphism UI, and real-time updates built using HTML, CSS, and JavaScript.

## 🚀 Features
- Real-time clock with AM/PM  
- Animated gradient background  
- Glassmorphism UI  
- Neon glow effect  
- Floating animation  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

## 🎯 Purpose
This project demonstrates advanced frontend UI design and JavaScript functionality.
